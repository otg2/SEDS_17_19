1. Make sure GAMA 1.6 is installed on your computer
2. Open up GAMA and right click on User models
3. Select Import and locate the FinalProject folder
		If this does not work, simply copy the FinalProject folder to 
		root/users/gama_workspace and refresh GAMA 
4. Open the Model folder and right click any version you would like to run
5. Select Run and select the experiment
